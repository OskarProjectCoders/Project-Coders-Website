In order to prepeare for your video-call appointment, please complete the following steps;

Create an illustration of how the project should look like for your users

For any text or images, which are to be included in the user interface, please send us an email with your text and the images labeled on your illustration and the product you send us.

Define text colors and background colors and write them in a #-format.

Define the font sizes for every piece of text.

Define where a link on a page should lead.

Please send us all this information before your video-call appointment, so that the video-call can be used as time effectively as possible and so the price can be set accordingly.
